# Final Project (Social Network) - The Odin Project - Full-Stack Ruby on Rails Course

Repo for the final project of the Full-Stack Ruby on Rails Course, from [The Odin Project](https://www.theodinproject.com/).

The objective was to build the core features of a social network like Facebook. The full project description and requirements can be found [here](https://www.theodinproject.com/lessons/ruby-on-rails-rails-final-project).

Features implemented:
- Sign up and sign in flows (including OAuth)
- Creating, editing and viewing profiles (profile pages)
- Sending friend requests
- Accepting and rejecting friend requests
- Posting texts
- Commenting on posts
- Liking posts
- Liking comments
- Real time notifications with WebSockets
